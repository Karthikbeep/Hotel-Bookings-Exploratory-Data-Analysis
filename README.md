# Hotel-Bookings-Exploratory-Data-Analysis

# Project Overview
This project performs Exploratory Data Analysis(EDA) on Hotel Bookings data to understand customer behavior, booking cancellations, seasonal trends, customer types and pricing patterns to help hotels mitigate revenue risks.

# Tools Used
1.Python
2.Pandas
3.Matplotlib
4.Seaborn

#Key Insights

1.Nearly 37% of bookings are canceled, posing a major revenue risk. 
2.City hotels experience higher cancellations than resort hotels. 
3.Long lead-time and high-ADR bookings show higher cancellation tendencies. 
4. May - August months have the highest cancellation volume.
5.Transient customers have the most cancellations.

# Conclusion

Overall, these insights can help hotels make better decisions around demand forecasting, cancellation policies, and customer segmentation to reduce revenue loss and improve operational efficiency.
